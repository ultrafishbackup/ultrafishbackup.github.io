2022 回顾与展望 | 2022 Review & Preview

Made by Mike_Zhang

Abstract

摘要

各位好。
时光飞逝，距离2021回顾已经过去一年了，又到了2022年度回顾的时刻。
今天是2022年最后一天，也是我来香港读书的第1218天，3年零4个月。按照惯例，用这篇文章来回顾和展望。
内容包括：

  • 年度回顾；
  • 目标检查；
  • 展望。

用一句话总结过去一年：

是特别的一年，渐入佳境，但有很多挑战。

1. Review

1. 年度回顾

2021.12

拾貳月

香港突然新增本地个案，使得维持了大半年的本地“零增长”被打断，由此开始了最严重的第五波疫情。

2022.1

壹月

在理大的第一个学期成绩公布了，取得了还算满意的成绩。

第五波疫情愈发严重，学校公布下个学期转为线上授课。

因政府的防疫政策，年三十晚餐不能在餐厅堂食。因此和室友在寝室里自己做了一顿年夜饭。

2022.2

貳月

第五波疫情发展极为快速，确诊数、死亡数快速上升，医疗系统的压力也越来越大，我们宿舍楼也出现越来越多的个案。

疫情愈发严重，最后还是选择“逃回”了家。

并且在隔离酒店度过这个特殊的学期。

2022.4

肆月

第一次参加谷歌的Code Jam算法比赛。很幸运的过了资格赛，但后来在第一轮淘汰了。

2022.5

伍月

收到了香港政府奖学金申请的结果，拿到了在理大的第一个奖学金。

第二个学期成绩公布了，一直担心疫情和隔离会影响成绩，但最后的结果还算满意，也要总结和改进。

收到了六月份的研究实习通知，是时候回港了。

2022.5-7

伍至柒月

很幸运，在疫情严重期间，度过了一个十分有意义的暑期。
参加了PolyHack2022 黑客马拉松比赛，社会服务学习，麻省理工研究实习，以及其他活动。
详情可见文章 2022暑期回顾 - Summer Break 2022 Review

2022.8

捌月

在新学期开始前，参加了理大新主校门的揭幕典礼，见证了新校门的启用。

然后就开始了大二的第一个学期。由于疫情有所缓解，理大恢复了全面线下授课模式，使我们重回校园。

2022.9

玖月

疫情反反复复，加上宿舍人数变多，宿舍的确诊个案也越来越多，我们宿舍楼也被列作强制检测的地点，需要核酸检测。

参加了去年我们部门组织的新生Python编程比赛 (The Freshman Python Programming Contest 2021/2022)的颁奖礼，认识了很多厉害的同学和老师。

2022.9-11

玖至拾壹月

经历了为期三个月的线下面授学期，相比与线上授课忙碌了不少，压力也变大了很多，但终于找回了校园学习的感觉。

这学期选了两门大三的课程，确实有不小的挑战。到最后一个月的时候，有大量的作业、小组讨论、项目等同时进行，是不小的挑战，但也十分充实。

在此期间，并没有更新很多的自学笔记文章，大部分还是每日“陶冶理操”的算法题汇总和课程笔记：

自学笔记：

Java面向对象编程之 封装性 | Java OOP Encapsulation

算法题汇总：

Problem Set ｜ 数据结构与算法题集

课程笔记：

Software Engineering Course Note
Database Systems Course Note
Mathematics I Course Note

2022.12

拾貳月

收到了香港理工大学奖学金申请结果，拿到了在理大第二个奖学金。

很不幸运，终究没逃过疫情的追击。因喉咙不舒服去校医就诊，医生给了口水瓶让我做核酸检测，交了样本的第二天收到了阳性的短信，需要上报，并去社区检疫中心隔离。

在检疫中心住了三晚四天，出现病症的第六天转阴了。好在病症较轻，一天发烧，多天喉咙痛，鼻塞，也没有其他的症状。吃了校医给的药也很快恢复了。另外，检疫中心环境不错，除了伙食不太好，但免费的就不说什么了。当然，不希望再去了。

大二第一学期正式结束，寒假期间开始第二学期课程预习和自学，总结成文章。
截止到本文章发布前，更新了相关文章：

Linux Learning 2 - File System
C Learning 2 - Operators & Control Statements
C Learning 1 - Data Type & IO
Linux Learning 1 - Bash Shell Basic

2022年12月26日晚
国家卫健委发布了《关于对新型冠状病毒感染实施“乙类乙管”的总体方案》。当中规定，2023年1月8日起，“取消入境后全员核酸检测和集中隔离。健康申报正常且海关口岸常规检疫无异常者，可放行进入社会面。”
换言之，明年1月8号后，香港就会和内地通关了，回内地不需要隔离了。从2020年2月8号开始的封关，终于在三年后解封了，实属不易。

2. Goals Check

2. 目标检查

以下是去年底在2021回顾一文中定下的目标。9项中完成了8项。

  • 保持长期健康及规律的饮食和睡眠。长期健康及规律的饮食和睡眠，辅以合适的运动与休息，是一切活动的基频(Fundamental Frequency)。
  • 保持日程规划与反思。
  • 保持高质量的学习。以原则为根本，以效率为基础，以质量为目标。
  • 多阅读专业课程相关书籍，学习相关项目。学习，研究，并总结为博客文章。
  • 提高博客更新频率。2022年至少更新30篇文章，多个开发性页面，继续维护网站。
  • 参加一项课外活动。
  • 拿到奖学金。
  • 免隔离回到内地。希望香港与内地尽快通关，回家不用隔离。
  • 进行年末总结并展望2023年。
