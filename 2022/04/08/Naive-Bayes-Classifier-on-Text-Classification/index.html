朴素贝叶斯文本分类器原理

2.1 In English

说人话：

  • 第一步，建立模型：
    • 整理出训练数据中所有语句中的词语；
    • 计算出每个词语在各类中出现的频率，即

    • P(wi∣cj)=某词在此类中出现的次数此类中所有词的个数P(w_i|c_j)=\frac{\text{某词在此类中出现的次数}}{\text{此类中所有词的个数}} P(wi​∣cj​)=此类中所有词的个数某词在此类中出现的次数​

    • 并且求出某类的初步判断概率，即

    • P(cj)=某类中所有语句的个数总的语句的个数P(c_j)=\frac{\text{某类中所有语句的个数}}{\text{总的语句的个数}} P(cj​)=总的语句的个数某类中所有语句的个数​

  • 第二步，应用模型：
    • 计算给定语句在各个类下的概率；
    • 计算语句ttt 在类ccc中的概率，即：

    • (初步判断概率)×(语句t中各个词语在类c中出现的频率的积)(\text{初步判断概率})\times (\text{语句$t$中各个词语在类$c$中出现的频率的积}) (初步判断概率)×(语句t中各个词语在类c中出现的频率的积)

    • =P(c)×∏wi∈tP(wi∣c)=P(c)\times \prod_{w_i\in t} P(w_i|c) =P(c)×wi​∈t∏​P(wi​∣c)

    • 概率数值最大的即为其所属类；

Detailed Explanation

详细解释：

1 Our Goal

1 我们的目标：

Based on the trained model, given a document (or a sentence/text), we can predict the class of the document.

ddd: a given document, or sentence/text;
CCC: set of all possible classes, e.g. (positive, negative, neutral);
ccc: the final result we want, to be the predicted class of ddd.
Goal is to get the maximum value of P(c∣d),c∈CP(c|d),c \in CP(c∣d),c∈C, which means given the document ddd, find its class with the maximum probability.

The goal: to get the maximum value of P(c∣d),c∈CP(c|d),c \in CP(c∣d),c∈C,

Based on Bayes’ Formula, we have,

P(c∣d)=P(d∣c)P(c)P(d)P(c|d)=\frac{P(d|c)P(c)}{P(d)} P(c∣d)=P(d)P(d∣c)P(c)​

So our final class ccc is

cMAP=argmaxc∈CP(c∣d)c_{MAP}=\mathrm{argmax}_{c\in C}P(c|d) cMAP​=argmaxc∈C​P(c∣d)

(MAP is maximum a posteriori = most likely class)

=argmaxc∈CP(d∣c)P(c)P(d)=\mathrm{argmax}_{c\in C}\frac{P(d|c)P(c)}{P(d)} =argmaxc∈C​P(d)P(d∣c)P(c)​

=argmaxc∈CP(d∣c)P(c)=\mathrm{argmax}_{c\in C}P(d|c)P(c) =argmaxc∈C​P(d∣c)P(c)

where we ignore P(d)P(d)P(d), because it is not related to ccc, like a constant.

=argmaxc∈CP(x1,x2,...,xn∣c)P(c)=\mathrm{argmax}_{c\in C}P(x_1,x_2,...,x_n|c)P(c) =argmaxc∈C​P(x1​,x2​,...,xn​∣c)P(c)

  • x1,x2,...,xnx_1,x_2,...,x_nx1​,x2​,...,xn​ are all words in ddd, e.g. all words in an message,
  • P(c),c∈CP(c),c\in CP(c),c∈C is the frequency of occurrence of this class, by count the relative frequencies, e.g. the frequencies of normal messages and spam messages.

For P(x1,x2,...,xn∣c)P(x_1,x_2,...,x_n|c)P(x1​,x2​,...,xn​∣c), we have two assumptions to simplify the prediction:

  • Bag of Words assumption: Assume words position doesn’t matter;
  • Conditional Independence: Assume P(xj∣cj)P(x_j|c_j)P(xj​∣cj​) are independent, which means each word in a message is independent with other words in the message.

Based on the assumption, we have:

P(x1,x2,...,xn∣c)=P(x1∣c)⋅P(x2∣c)⋅...⋅P(xn∣c)=∏x∈{x1,x2,...,xn}P(x∣c)P(x_1,x_2,...,x_n|c)=P(x_1|c) \cdot P(x_2|c) \cdot ... \cdot P(x_n|c)=\prod_{x\in \{x_1,x_2,...,x_n\}}P(x|c) P(x1​,x2​,...,xn​∣c)=P(x1​∣c)⋅P(x2​∣c)⋅...⋅P(xn​∣c)=x∈{x1​,x2​,...,xn​}∏​P(x∣c)

Therefore, we can simplify the prediction cMAPc_{MAP}cMAP​ above:

cMAP=argmaxc∈CP(c)∏x∈XP(x∣c),X={x1,x2,...,xn}c_{MAP}=\mathrm{argmax}_{c\in C}P(c)\prod_{x\in X}P(x|c),\\ X=\{x_1,x_2,...,x_n\} cMAP​=argmaxc∈C​P(c)x∈X∏​P(x∣c),X={x1​,x2​,...,xn​}

As same as:

cMAP=argmaxcj∈CP(cj)∏i∈positionsP(xi∣cj)c_{MAP}=\mathrm{argmax}_{c_j\in C}P(c_j)\prod_{i\in positions}P(x_i|c_j) cMAP​=argmaxcj​∈C​P(cj​)i∈positions∏​P(xi​∣cj​)

positionspositionspositions = all word positions in the test document.

Multiplying floating point numbers may cause underflow loss,
then based on,

log⁡(ab)=log⁡(a)+log⁡(b)\log(ab)=\log(a)+\log(b) log(ab)=log(a)+log(b)

then,

cMAP=argmaxcj∈C[log⁡P(cj)+∑i∈positionslog⁡P(xi∣cj)]c_{MAP}=\mathrm{argmax}_{c_j\in C}\Biggl[\log{P(c_j)}+\sum_{i\in positions}\log{P(x_i|c_j)}\Biggl] cMAP​=argmaxcj​∈C​[logP(cj​)+i∈positions∑​logP(xi​∣cj​)]

This is our final goal, next we need to get the value of each term.
