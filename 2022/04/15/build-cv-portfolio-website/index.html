3 Build CV or Portfolio Website

搭建个人简历或作品集网站

3.1 Build CV website by PDF

通过PDF文档搭建简历网站

这种方式是最简单，也是最常用的，可以把现有的PDF简历或者作品集文档搭建成网站。

1 准备好你的个人简介PDF文档；

  • 如demo.pdf；

2 新建一个index.html文件；

  • 写入以下代码：
    • 其中，可以修改My personal CV为你的个人简介的标题；
    • demo.pdf为你的个人简介PDF文档；
    • demo.pdf与index.html应在同一文件夹内；
    • 下载示例代码：https://github.com/zhangwengyu999/pdf-cv-demo

3 部署至GitHub仓库；

  • 新建仓库
    • 进入GitHub的个人空间；
    • 点击Repositories；
    • 点击New；

  • 填写仓库信息；

    • 细心填写 Repository name，这是之后访问你的地址；
    • 选择 Public；
    • 点击 Create repository 完成；

  • 上传文件至仓库；

    • 上传前面的文件（如demo.pdf与index.html）；
    • 填写Commit信息；
    • 点击Commit changes；

  • 使用GitHub Pages；

    • 点击上方 Settings；
    • 点击左侧 Pages；
    • 选择Brach为main；
    • 点击Save；

  • 可以看到提示框显示部署成功，访问此地址即可看到你的网站（第一次访问等待时间会稍长，请多次刷新）；

    • 访问示例网站：https://zhangwengyu999.github.io/pdf-cv-demo/

3.2 Build CV website by Markdown

通过Markdown文档搭建简历网站

让给你会使用Markdown，可以用Markdown编写简历，并导出为html文档（例如使用VS Code）。重复前面部署至github仓库的步骤，把导出的html文件重命名为index.html后，再上传至一个新的仓库。

  • 下载示例代码：https://github.com/zhangwengyu999/md-cv-demo
  • 访问示例网站：https://zhangwengyu999.github.io/md-cv-demo/
