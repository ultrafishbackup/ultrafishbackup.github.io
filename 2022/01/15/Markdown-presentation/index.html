使用Markdown制作演示文稿 Slidev

以上视频是我对之前文章-有符号二进制数表示方法Signed binary number representation中一部分内容的演示。

这用到了一款基于reveal.js框架的reveal-md工具，可以使用Markdown来生成演示文稿（类似于PPT）。

reveal-md GitHub主页

Usage

使用方法：

  • 参考reveal-md GitHub主页，使用命令行运行：

安装：
npm install -g reveal-md

运行：
reveal-md slides.md
（slides.md为目标Markdown文件）

随后可用浏览器打开本地端口进行演示。

  • 使用VS Code插件运行：

vscode-reveal插件

可实时渲染：

Feature

特点：

  • 使用Markdown编写，可用---分隔每页演示文稿，或自定义分隔符；
  • 使内容与排版分离，实时渲染演示文稿，能让创作者更加专注于表达的内容；
  • 有多种主题可以使用；
  • 支持HTML和CSS，可高度自定义页面样式；
  • 可导出为PDF文档。

引用:
reveal-md - GitHub

写在最后

reveal-md工具十分方便，可快速且精简的生成演示文稿，但是其只能做简单的演示，不能进行稍复杂的排版。

但是一款名为Slidev的演示文稿工具更加专业更加强大，专为开发者设计，之后会用文章记录。

最后，希望大家一起交流，分享，指出问题，谢谢！
