1 Binary Tree Traversal

1.1 Pre-order Traversal

First visit the root, then traverse the left subtree, finally traverse the right subtree.

Binary Tree Preorder Traversal - LeetCode Problem #144

1.2 In-order Traversal

First traverse the left subtree, then visit the root, finally traverse the right subtree.

Binary Tree Inorder Traversal - LeetCode Problem #94

1.3 Post-order Traversal

First traverse the left subtree, then traverse the right subtree, finally visit the root.

Binary Tree Postorder Traversal - LeetCode Problem #145

1.5 Level Order Traversal
