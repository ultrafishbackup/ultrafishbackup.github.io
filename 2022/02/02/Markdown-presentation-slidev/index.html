使用Markdown制作演示文稿 Slidev

以上视频是我对之前文章-浮点数二进制数表示方法 Floating point numbers representation中一部分内容的演示。

从视频最后可以发现，此演示视频也是使用Markdown编写的，但用到的工具不同于上一篇文章中的reveal-md，

而是用到了用到了一款名为Slidev的演示文稿工具，专为开发者设计，功能十分强大。支持Markdown语法，可更改主题，可定制主题，支持代码片段展示，以及各种高自定义功能。

我会简单介绍一些实用的语法，Slidev的功能十分多，具体用法请参考Slidev 主页。

2. Layouts

页面布局

具体内容请参考：布局-Slidev

  • 用---分隔每页演示文稿

2.1 Cover

封面页

  • 第一页默认为封面页；
  • 居中显示；
  • 根据Markdown标题层级大小显示。

2.2 Default

基础页

  • 默认靠左显示；
  • 按照Markdown语法显示；
  • 若段落间行距较小，可用<br>换行符。

2.3 Center

居中页

  • 需要在段落开始处标明layout属性为center;
  • 类似于封面页；
  • 文字默认于中部靠左对齐；
  • 使用class: text-center可使文字居中对齐。

2.4 Image

图片页

  • 需要在段落开始处标明layout属性，可为image,image-left,image-right，使图片处在不同的位置；
  • layout属性为图片链接，可以是在线图片链接，也可以是本地图片。
    • 使用本地图片前，需要在工作目录下新建public文件夹，并把图片资源放在此文件夹中，图片链接默认从public文件夹中访问）。

2.5 Iframe

网页片段页

  • 与图片页类似，可设置网页所在的相对位置；
  • 设置url属性标明网址；

3. Code Block

代码片段

具体内容请参考：代码块-Slidev

  • 使用 Markdown 语法编写代码块；
  • 可在开头使用{}标注高亮的代码。可用|隔开，设置每次点击分别高亮的代码

4. Animations

动画

具体内容请参考：动画-Slidev

4.1 元素显示动画

  • 使用<v-click> </v-click>标记点击后出现的元素；
  • 使用<v-after> </v-after>标记点击前出现的元素。

4.2 物件运动动画

  • 使用v-motion标记，标明初始状态和最终状态，自动生成动画；
  • 可控制x,y坐标，不透明度等属性；
  • 可设置延时，时长等属性；
  • 具体内容请参考：动画-Slidev。

4.3 箭头标记

  • 使用<arrow />标记箭头；
  • 可设置头尾坐标，颜色，宽度，箭头大小等属性。
